아직 코드 구현 전 단계다. `docs/CLAUDE_CODE_EXECUTION_PROMPTS.md`의 Prompt 0부터 순서대로 실행하여 구현을 진행한다. 각 단계마다 테스트와 git commit을 수행한다.
